---
title: "API reference"
---

Claptrap exposes a REST API at `/api/v1` for managing all catalog
resources. See the [API architecture](/architecture/api-and-mcp) page
for design details.

## Base URL

```
https://docs.claptrap.dev/api/v1
```

## Resources

| Resource | Description |
| --- | --- |
| [Entries](/api-reference/entries/list-entries) | Normalized content records |
| [Sources](/api-reference/sources/list-sources) | Upstream content origins |
| [Sinks](/api-reference/sinks/list-sinks) | Delivery targets |
| [Subscriptions](/api-reference/subscriptions/list-subscriptions) | Tag-based routing rules |

## Response format

All responses are JSON. Successful responses return the resource or
list of resources directly. Error responses include a structured error
object.
